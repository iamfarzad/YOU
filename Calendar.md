## Temporal System

Track events across time with context for pattern recognition.

### Entry Format
```
YYYY-MM-DD
- Category: [[Category]]
- Event: [Description]
- Energy: [+/-/=/++/--]
- Connections: [[Related aspect]]
- Perspective: [Insight/Concern/Aspiration/Observation]
```

### Time Dimensions

**Past Context**
- Historical precedents
- Seasonal patterns
- Recurring cycles
- Anniversary effects

**Current Moment**
- Immediate situation
- Active stressors
- Available resources
- Energy state

**Future Projection**
- Implications of today
- Compounding factors
- Intervention windows
- Connection points yet to surface
