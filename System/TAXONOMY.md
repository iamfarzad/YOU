# Taxonomy (Hidden Scaffolding)

The taxonomy is a map for the AI, not a form for the user.
It stays mostly hidden from the default user experience and only surfaces categories when evidence warrants it.

## Taxonomy Philosophy
- Use categories as latent slots for retrieval and linking.
- Surface only relevant, active, source-linked categories.
- Keep hypotheses separate from confirmed facts.
