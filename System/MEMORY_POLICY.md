# Memory Write Policy

| Memory type | Auto-write? | User confirmation |
|---|---:|---:|
| Raw note | yes | no |
| Project fact | yes | optional |
| User preference | yes | optional |
| Profile identity fact | no | yes |
| Health/wellbeing hypothesis | no | yes |
| Relationship hypothesis | no | yes |
| Future hypothesis | yes | review queue |
| Sensitive category activation | no | yes |
| Deletion | no | yes |
