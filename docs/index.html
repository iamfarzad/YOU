How it works

From signal to insight, automatically

Four layers. Raw signals go in, patterns come out. Your agent handles structure. You provide content.

01

Capture

Raw signals land in Inbox — journal entries, observations, any text. Immutable.

02

Normalize

Agent extracts events, assigns categories, timestamps. Conservative, no hallucination.

03

Compile

Update category state, cross-link domains, update awareness snapshot and pattern log.

04

Surface

Categories appear only when evidence thresholds are met. Hidden until earned.

Agent-agnostic

One memory layer. Any agent.

YOU is just markdown files and a CLI. Any agent that can read and write files — Claude, GPT, Hermes, OpenClaw, a shell script — can use it. No API keys, no vendor lock-in, no cloud dependency.

Get started

Three commands. That's it.

No config, no accounts, no cloud. Clone, capture, compile.

bash Capture a signal 
# Your agent captures a raw observation
you capture --source journal --text "Missed deadline
at work, couldn't sleep, short with family"

# YOU stores it immutable in Inbox/raw/
# Then normalizes: Work(--), Sleep(--), Family(--)
you process-inbox
bash Query + recall 
# Ask about a life domain
you query --category "Work"

# Agent reads compressed awareness first
you recall --query "burnout patterns"

# Pattern surfaces:
# Work stress → 3d → Sleep ↓
# Sleep ↓ → 5d → Patience ↓
# Confidence: 0.78 · Evidence: 4 entries
What makes it different

Built for real life, not a demo

Local-first, always

All data stays on your machine. Markdown files in a git repo. No cloud, no accounts, no API keys to leak. Your life data never leaves your disk.

Hidden until earned

51 life categories exist as scaffolding, but stay hidden until evidence thresholds are met. A category surfaces only when you've logged enough real signals to justify it. No empty folders.

Cascade detection

Work stress doesn't stay at work. YOU tracks cross-domain effects: how one domain's decline predicts another's. The system connects A and Z even when their link is invisible to you.

Temporal tracking

Every entry carries Past, Current, and Future. Not just "what happened" but "what happens next if nothing changes, and what if something does?"

Agent-maintained

Humans provide content. Agents handle structure: filing, cross-linking, pattern detection, awareness compression. You think. The agent files.

Plain markdown

No database, no binary format. Every file is readable markdown. Edit in Obsidian, VS Code, or cat. Version with git. It's yours, permanently.

Philosophy

Life is a system

What happens here if nothing changes?
And what if something does?

— The question at the heart of every YOU entry.

YOU was built from the recognition that life domains don't exist in isolation. Your work stress affects your sleep. Your sleep affects your patience. Your patience affects your relationships. Your relationships affect your career decisions. Most tools track one domain. YOU tracks the connections between all of them.
